# 概述

## 什么是计算机组成原理

计算机组成原理是研究计算机硬件系统基本组成和工作原理的学科。它从硬件的角度解释计算机如何执行程序、存储数据以及与外部设备交互。

## 学习目标

- 理解计算机系统的层次结构
- 掌握数据的各种表示方法
- 理解指令系统的设计原理
- 掌握CPU的工作原理
- 理解存储层次结构
- 了解总线与I/O系统

## 为什么学习计算机组成原理

1. **深入理解计算机**：了解计算机底层工作原理，有助于编写更高效的程序
2. **性能优化**：理解硬件特性可以指导软件优化
3. **系统设计**：为系统级编程和架构设计打下基础
4. **面试准备**：计算机组成原理是技术面试的常见考点

## 学习路径

1. 数据表示 → 2. 指令系统 → 3. CPU设计 → 4. 存储系统 → 5. I/O系统
