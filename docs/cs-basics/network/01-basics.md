# 网络基础

## 什么是计算机网络

计算机网络是指将地理位置不同的、具有独立功能的多台计算机及其外部设备，通过通信线路连接起来，在网络操作系统，网络管理软件及网络通信协议的管理和协调下，实现资源共享和信息传递的计算机系统。

## 网络的分类

### 按覆盖范围分类

1. **局域网（LAN）**
   - 覆盖范围：几米到几公里
   - 例子：家庭网络、办公室网络、校园网

2. **城域网（MAN）**
   - 覆盖范围：几公里到几十公里
   - 例子：城市范围内的网络

3. **广域网（WAN）**
   - 覆盖范围：几十公里到几千公里
   - 例子：互联网

### 按拓扑结构分类

1. **总线型**：所有设备连接到同一条总线
2. **星型**：所有设备连接到中心设备
3. **环型**：设备首尾相连形成环
4. **网状型**：设备之间有多条连接路径

## 网络性能指标

1. **带宽**：网络传输速率
2. **延迟**：数据传输的时间延迟
3. **吞吐量**：单位时间内传输的数据量
4. **丢包率**：传输过程中丢失的数据包比例
